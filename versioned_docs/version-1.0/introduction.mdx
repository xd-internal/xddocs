# Full Observability. Zero Vendor Lock-in.

XplurData is a self-hosted, Docker-based observability platform built for teams who want full control over their telemetry — no per-seat pricing, no data leaving your infrastructure.

## One-line Install

```bash
# One command — full stack, no cloud required
$ bash -c "$(curl -fsSL \
    https://raw.githubusercontent.com/xd-internal/oss-stack/main/install.sh)"
```

## How It Works

| | |
|---|---|
| ⚡ **Ingest** | Receive logs, traces and metrics from any service via OpenTelemetry OTLP/HTTP on port 4318. |
| 🗄 **Store** | Apache Doris v4.0.4 — columnar analytical database purpose-built for telemetry at scale. |
| 🔍 **Explore** | XD-API + React UI with full-text search, filtering, and real-time queries at port 80. |
| 🐳 **Deploy** | 4 Docker containers. One install command. Runs on Linux, macOS or Windows in under 5 minutes. |
